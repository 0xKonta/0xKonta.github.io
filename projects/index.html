Cyber Threat Intelligence

Project Introduction

I am currently working on several sub-projects related to Cyber Threat Intelligence. The goal is to create tools to help users and myself track publications and activities related to cyber threats.

Ongoing and Upcoming Projects

  • Development of various monitoring tools, each targeting a specific aspect of CTI.
  • Setting up accounts on various platforms (Mastodon, X.com, Telegram) to disseminate notifications.

These projects will allow users to choose their preferred platform to receive updates on cyber threats.

Leaks

Access CTI-Leaks

CTI-Leaks is one of the first tools developed as part of this project. It allows monitoring of new files that have leaked on the internet, particularly by ransomware groups.

  • An export button allows retrieving all data in JSON format, facilitating the use and analysis of collected information.

Other projects are under development to complete this suite of CTI tools.
